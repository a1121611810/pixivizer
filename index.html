Pictelio

核心功能

Pictelio 提供流畅的插画浏览体验，所有数据均通过 Pixiv 官方 API 获取并在本地处理。

推荐与关注 feed

快速切换「推荐」与「关注」标签，沉浸式浏览 Pixiv 每日精选与你关注的创作者动态。

作品详情与大图浏览

查看单页或多页作品，支持缩放、滑动翻页与 Ugoira 动图播放，还原插画细节。

年龄门与 R-18/R-18G 过滤

内置年龄确认与分级内容过滤，帮助你在应用设置中管理敏感内容的显示权限。

举报与屏蔽

遇到不适内容？一键举报或屏蔽用户/标签，打造更符合个人偏好的浏览环境。
