# 系列弹出面板术语表

| 术语 | 定义 |
|------|------|
| **SeriesSheet** | 从屏幕底部弹出的面板组件，展示系列（series）内所有小说条目。通过 `@capacitor/app` 的轻扫手势可关闭。 |
| **SeriesSheetItem** | SeriesSheet 中的单篇小说条目行，包含封面缩略图、标题、字数、标签。 |
| **系列（Series）** | Pixiv 上的作品系列概念，作者可将多篇相关小说（如连载章节）组织为一个系列。系列包含元数据（标题、作者、总字数）和小说列表。 |
| **系列导航** | 在小说详情页底部固定的导航栏和 SeriesSheet 之间切换小说的交互模式。 |
| **IllustTags** | 可复用的标签展示组件，将标签数组渲染为一系列 chip 样式的 `<span>` 元素。支持 `small` / `medium` 两种尺寸。 |
| **标签行** | SeriesSheetItem 中展示标签的独立行，位于字数行下方，使用 `flex-wrap` 自然换行布局。 |
