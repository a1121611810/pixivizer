# Pixiv 第三方客户端 - 产品需求文档 (MVP)

> 版本: 1.0  
> 日期: 2026-02-23  
> 状态: 待开发

---

## 1. 产品概述

### 1.1 产品定位

第三方 Pixiv 桌面/安卓客户端，提供插画、小说浏览、搜索、下载等功能。

### 1.2 目标用户

- Pixiv 重度用户
- 需要桌面/大屏浏览体验的用户
- 需要下载保存插画/小说的用户

### 1.3 平台支持

- [x] Windows
- [x] macOS
- [x] Linux
- [x] Android (APK)

---

## 2. 技术架构

### 2.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 框架 | Tauri 2.x | 跨平台桌面/移动端框架 |
| 前端框架 | Vue 3 | 渐进式前端框架 |
| UI 组件库 | Varlet UI | Material Design 风格 |
| 状态管理 | Pinia | Vue 官方推荐状态库 |
| 语言 | JavaScript | 纯 JS，无 TypeScript |
| 后端 | Rust | Token 存储、API 代理 |

### 2.2 架构图


## 3. 功能需求

### 3.1 功能优先级

| 优先级 | 功能 | 描述 | 状态 |
|--------|------|------|------|
| P0 | OAuth 登录 | Pixiv 账号 OAuth 认证登录 | 待开发 |
| P0 | 首页推荐 | 热门插画、轮播图展示 | 待开发 |
| P0 | 排行榜 | 日榜、月榜、周榜、原创、新人 | 待开发 |
| P0 | 插画详情 | 大图查看、作者信息、相关推荐 | 待开发 |
| P1 | 关键词搜索 | 按关键词搜索插画/小说 | 待开发 |
| P1 | 标签搜索 | 按标签搜索插画 | 待开发 |
| P1 | 小说列表 | 小说排行榜、标签筛选 | 待开发 |
| P1 | 小说阅读 | 小说阅读器、分页 | 待开发 |
| P1 | 图片下载 | 保存插画原图到本地 | 待开发 |
| P1 | 小说下载 | 保存小说文本到本地 | 待开发 |
| P2 | 用户收藏 | 收藏/取消收藏插画 | 待开发 |
| P2 | 关注作者 | 关注/取消关注作者 | 待开发 |
| P2 | 查看收藏 | 查看自己收藏的插画 | 待开发 |

### 3.2 详细功能说明

#### 3.2.1 认证模块 (P0)

- **OAuth 登录**
  - 打开 Pixiv 授权页面
  - 回调获取 access_token / refresh_token
  - Token 存储在 Rust 层（安全）
  - 自动 Token 刷新

- **登录状态**
  - 显示用户头像、用户名
  - 未登录引导登录

#### 3.2.2 浏览模块 (P0)

- **首页推荐**
  - 热门插画列表
  - 轮播图展示推荐内容
  - 下拉刷新、上拉加载更多

- **排行榜**
  - 日榜 (Daily)
  - 周榜 (Weekly)
  - 月榜 (Monthly)
  - 原创 (Original)
  - 新人 (Rookie)
  - 标签筛选

#### 3.2.3 插画详情 (P0)

- **图片展示**
  - 原图查看
  - 缩放、滑动浏览
  - 动图播放

- **作品信息**
  - 标题、简介
  - 标签
  - 上传日期
  - 浏览数、收藏数、评论数

- **作者信息**
  - 头像、用户名
  - 关注按钮
  - 进入作者主页

#### 3.2.4 搜索模块 (P1)

- **关键词搜索**
  - 插画搜索
  - 小说搜索
  - 搜索历史记录

- **标签搜索**
  - 热门标签
  - 标签联想

#### 3.2.5 小说模块 (P1)

- **小说列表**
  - 排行榜
  - 标签筛选
  - 排序方式

- **小说阅读**
  - 文本阅读器
  - 分页/滚动
  - 字体大小调节
  - 背景主题（明亮/黑暗）

#### 3.2.6 下载模块 (P1)

- **图片下载**
  - 下载原图
  - 选择保存路径
  - 进度显示

- **小说下载**
  - 导出为 TXT
  - 选择保存路径

#### 3.2.7 用户模块 (P2)

- **收藏功能**
  - 收藏/取消收藏
  - 查看收藏列表

- **关注功能**
  - 关注/取消关注作者
  - 查看关注列表

## 5. 非功能性需求

### 5.1 性能要求

- 页面加载时间 < 3秒
- 图片懒加载
- 列表虚拟滚动（长列表优化）

### 5.2 兼容性

- Windows 10+
- macOS 10.15+
- Android 7.0+

### 5.3 安全要求

- Token 存储在 Rust 后端，不暴露在前端
- HTTPS 加密传输

## 7. 风险与依赖

### 7.1 主要风险

1. **Pixiv API 变更** - 官方可能修改 API
2. **CORS 限制** - 通过 Rust 代理解决
3. **登录方式** - 需要保号，OAuth 可能被限制

### 7.2 依赖库

- 前端: Varlet UI, Pinia, vue-router
- Rust: reqwest, tauri, serde
