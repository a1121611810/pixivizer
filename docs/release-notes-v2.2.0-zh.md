# v2.2.0

## 推荐流

- 新增「综合 / 插画 / 漫画」子标签切换
- 切换标签自动保留滚动位置
- 防止快速切换导致的请求竞态

## 图片源代理

- 设置中可配置多个上游图片源
- Race 模式并发探测，自动选择最快 CDN
- Single 模式固定使用指定源
- 后台缓存过期自动刷新

## DNS over HTTPS

- Android 实验性 DoH 开关
- 基于 OkHttp 和自定义 PictelioHttpPlugin
- Web 端自动禁用

## 设置面板

- 底部弹出改为左侧抽屉，更符合 Fluent 设计
- 各配置区域间距统一

## 骨架屏

- 初始加载时显示占位卡片
- 自动适配布局模式间距

## 网格与单列

- 网格模式专用 GridCard，显示头像、用户名和关注按钮
- 修复网格行高计算错误导致的卡片重叠
- 修复单列 Y 轴位置公式错误

## 性能

- 瀑布流布局计算迁移到 Web Worker
- 代理请求延迟降低
- 布局切换不再触发冗余刷新

## Bug 修复

- 图片源主开关与 UI 状态可靠同步
- Fluent Web Component 兼容性修复
- Radio group 编程赋值不同步修复
- 布局切换清除旧瀑布流缓存
- GridCard 样式与 ImageCard 统一
- 下拉刷新丢失滚动位置修复
- imageHostStore 持久化与响应式修复
